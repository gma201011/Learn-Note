## 如何隱藏一個元素

|                                    | 佔位 | 點擊事件 |
| ---------------------------------- | ---- | -------- |
| `display: none`                    | X    | X        |
| `opacity: 0`                       | O    | O        |
| `visibility: hidden`               | O    | X        |
| `clip-path: circle(0)`             | O    | X        |
| `position: absolute; top: -999px;` | X    | O        |
